# PolyTrack Replay Mod

A cinematic replay and camera control mod for PolyTrack, inspired by the Minecraft Replay Mod. This project focuses on giving players full control over camera movement, angles, and recording workflows to create smooth, professional-looking replays and edits.

---

## Overview

PolyTrack Replay Mod allows you to record gameplay, then replay it with advanced camera tools such as keyframes, custom camera rigs, smoothing, and cinematic effects.

The goal is to turn PolyTrack into a tool not just for racing, but for content creation.

---

## Features

- Keyframe-based camera editor  
- Additional camera types  
- Cinematic camera settings (FOV, depth of field, etc.)  
- Smooth camera movement  

---

## Planned Workflow

1. Select a run  
2. Enter replay mode (watch)  
3. Build the camera path using the keyframe editor  
4. Press record to play back the sequence  
5. Choose export settings (codec, resolution, FPS, etc.)  
6. Download the final video  

---

## Design Goals

- Smooth, intuitive UI (inspired by Blender timeline)  
- High flexibility for creators  
- Minimal impact on core gameplay  
- Expandability for future features
